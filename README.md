# Smart Business Consulting — Neuromarketing Demo Site

## How to host on GitHub Pages
1) Create a **public** GitHub repository (e.g., `neuromarketing-site`).
2) Upload these files directly (drag-drop in the web UI) or push via Git.
3) Go to **Settings → Pages**. Under "Build and deployment", choose:
   - Source: **Deploy from a branch**
   - Branch: **main** (or `master`) and folder: **/** (root)
4) Save. Your site will be served at `https://USERNAME.github.io/REPO` within a minute.
5) Submit your live URL with your blog/report.

## Pages
- `/index.html` — Home (value proposition, primary CTA)
- `/about.html` — Principles & value
- `/services.html` — Offers with CTAs
- `/testimonials.html` — Social proof
- `/contact.html` — Form + brief privacy/ethics note
